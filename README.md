# react-native-azure-maps

React-Native wrapper for Azure Maps iOS and Android SDKs

## Installation

```sh
npm install react-native-azure-maps
```

## Usage

```js
import { multiply } from 'react-native-azure-maps';

// ...

const result = await multiply(3, 7);
```
